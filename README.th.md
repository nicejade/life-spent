> พื้นที่เงียบ ๆ ไว้ใช้มองเวลาอย่างตรงไปตรงมา ไม่มีคำตัดสิน ไม่มีคำสั่งสอน มีเพียงตัวเลขเท่านั้น

**LifeSpent** เป็นเว็บแอปมินิมอลที่บอกให้คุณรู้ว่า “ตอนนี้คุณใช้ชีวิตไปแล้วกี่ส่วน” จากอายุปัจจุบัน อายุขัยเฉลี่ย และอายุมัธยฐานของประชากร จุดประสงค์ไม่ใช่การสร้างความกลัวหรือเร่งรัดให้ต้องทำอะไรเพิ่ม แต่เป็นการช่วยให้คุณเห็นชัดขึ้นว่า เวลาที่เหลืออยู่ไม่ได้ยาวนานอย่างที่เรามักคิดไว้เสมอไป

### ✨ จุดเด่นสำคัญ

- **ค่าเริ่มต้น** : อายุขัยเฉลี่ย 75.37 ปีสำหรับผู้ชาย และ 80.88 ปีสำหรับผู้หญิง; อายุมัธยฐานของประชากรโลก 31.1 ปี (สามารถปรับแต่งได้ในโค้ด)
- **โทน** : สงบ สุขุม เรียบง่าย – ไม่ใช่เครื่องมือสร้างแรงบันดาลใจ และไม่ใช่ตัวนับถอยหลังชวนกดดัน
- **ความเป็นส่วนตัว** : ไม่มีระบบบัญชี ไม่มีการติดตาม ไม่เก็บข้อมูลส่วนตัว เกิดขึ้นแค่ “การคำนวณ”

### 🕊️ ปรัชญา

LifeSpent ไม่ได้ถูกสร้างมาเพื่อให้คุณ “ต้องทำอะไรเพิ่มเดี๋ยวนี้”  
แต่มันคือกระจกบานเล็ก ๆ ที่สะท้อนให้เห็นเวลาที่ผ่านไป และเวลาที่เหลืออยู่ อย่างตรงไปตรงมา

- **ข้อมูลเชิงวัตถุวิสัย** : ใช้ข้อมูลคณิตศาสตร์ประกันภัยจริง (อายุขัยเฉลี่ย: ผู้ชาย 75.37 ปี, ผู้หญิง 80.88 ปี; อายุมัธยฐานของประชากรโลกโดยค่าเริ่มต้น: 31.1 ปี)
- **การออกแบบแบบมีพิธีรีตองเล็กน้อย** : อินเทอร์เฟซโทนมืด ทันสมัย ให้ความรู้สึกนิ่ง ละมุน เน้นตัวเลขมากกว่าลูกเล่น
- **ประสบการณ์แบบไร้แรงเสียดทาน** : ไม่ต้องสมัคร ไม่ต้องล็อกอิน ไม่ส่งข้อมูลไปที่ไหน ตัวเลขทั้งหมดอยู่บนหน้าจอของคุณเท่านั้น

### 🛠️ เทคโนโลยีที่ใช้

ออกแบบโดยคำนึงถึงประสิทธิภาพ ความปลอดภัยด้านชนิดข้อมูล และมาตรฐานสมัยใหม่

- **Frontend** : [Svelte 4](https://svelte.dev/)
- **ภาษา** : [TypeScript](https://www.typescriptlang.org/)
- **สไตล์** : [Tailwind CSS](https://tailwindcss.com/)
- **เครื่องมือ build** : [Vite](https://vitejs.dev/)

### 🚀 เริ่มต้นใช้งาน

#### สิ่งที่ต้องมี

- Node.js (แนะนำเวอร์ชัน LTS ล่าสุด)
- npm หรือ pnpm

#### วิธีติดตั้งและรันโปรเจกต์

```bash
# โคลนโปรเจกต์
git clone https://github.com/nicejade/life-spent.git
cd life-spent

# ติดตั้ง dependencies
npm install

# รันเซิร์ฟเวอร์สำหรับพัฒนา
npm run dev

# สร้าง build สำหรับ production
npm run build
```

### 🌐 การเข้าถึง

- **โหมดพัฒนา (local)** : รัน `npm run dev` แล้วเปิด `http://localhost:5173` ในเบราว์เซอร์
- **โหมด production** : รัน `npm run build` แล้วนำโฟลเดอร์ `dist/` ไป deploy บนโฮสต์แบบ static
- **ตัวเลือกโฮสต์** : ใช้ได้กับบริการ static ทั่วไป เช่น Vercel, Netlify, GitHub Pages เป็นต้น

### 📂 โครงสร้างโปรเจกต์

```bash
src/
├── helper/            # ลอจิกหลัก ค่าคงที่ และยูทิลิตี
├── types/             # คำจำกัดความชนิดข้อมูลของ TypeScript
├── components/        # คอมโพเนนต์ UI ของ Svelte
├── assets/icons/      # ไอคอน SVG (check, github, moon, selector, sun)
├── styles/            # สไตล์ส่วนกลางปริมาณน้อย
├── App.svelte         # จุดเริ่มต้นของแอปพลิเคชัน
└── main.ts            # สคริปต์บูตระบบ
```

### 🎨 แนวทางการออกแบบ

- **ความเรียบง่าย** : ข้อความสั้น กระชับ ชัดเจน เปิดพื้นที่ให้ความรู้สึกของผู้ใช้เอง
- **ความโปร่งใสแบบกระจกฝ้า** : การ์ดโปร่งแสงและเลเยอร์ที่นุ่มนวล ให้ความรู้สึกโมเดิร์นแต่ไม่วุ่นวาย
- **การเข้าถึง (Accessibility)** : ใช้ HTML แบบ semantic และสนับสนุน ARIA ตั้งแต่แรกเริ่ม
- **รองรับหลายหน้าจอ** : ออกแบบแบบ mobile-first ให้คุณหยุดดูเวลาได้อย่างรวดเร็วไม่ว่าคุณจะอยู่ที่ไหน

### 🤝 การมีส่วนร่วม

ยินดีรับการมีส่วนร่วมที่ช่วยเสริมโทนสงบ เรียบง่าย และความมินิมอลของโปรเจกต์นี้

- **หากต้องการเปลี่ยนแปลงครั้งใหญ่** : กรุณาเปิด issue เพื่อพูดคุยแนวทางก่อน
- **เวิร์กโฟลว์** : fork โปรเจกต์ สร้าง branch จาก `main` แล้วส่ง PR ที่มีขอบเขตชัดเจน
- **การจัดโครงสร้างโค้ด** : ลอจิกและการคำนวณควรอยู่ใน `lib/` ขณะที่ UI อยู่ในคอมโพเนนต์ `src/`
- **เมื่อมีการเปลี่ยนแปลงลอจิก** : หากทำได้ให้เพิ่ม test หรือออกแบบให้เขียน test ภายหลังได้ง่าย
- **สไตล์โค้ด** : เน้นใช้ยูทิลิตีของ Tailwind และหลีกเลี่ยงการเขียน CSS ปริมาณมากเอง
