> Спокойный взгляд на время. Без оценок, без морализаторства — только цифры.

**LifeSpent** — это минималистичное веб‑приложение, которое показывает, какая часть вашей жизни уже позади, опираясь на ваш возраст, среднюю ожидаемую продолжительность жизни и медианный возраст населения. Цель не в том, чтобы напугать или замотивировать, а в том, чтобы мягко разрушить иллюзию «бесконечной молодости» и напомнить: оставшееся время — не бесконечно.

### ✨ Основные моменты

- **Значения по умолчанию**: средняя ожидаемая продолжительность жизни — 75,37 года для мужчин и 80,88 года для женщин; медианный возраст населения мира — 31,1 (может быть изменён в коде).
- **Тональность**: спокойная, сдержанная, рефлексивная — без пафоса и без давления.
- **Конфиденциальность**: без регистрации, без аккаунтов, без трекинга — только расчёты на вашей стороне.

### 🕊️ Философия

LifeSpent — это не мотивационный сервис и не мрачный «счётчик до конца жизни».  
Это скорее зеркало, в котором отражается ваша траектория во времени.

- **Объективные данные**: используются реальные актуарные данные (средняя ожидаемая продолжительность жизни: 75,37 года для мужчин и 80,88 года для женщин; медианный возраст населения мира по умолчанию: 31,1 года).
- **Церемониальный дизайн**: тёмный, современный интерфейс, который подчёркивает главное — числа и ощущение времени.
- **Никакого трения**: ничего не нужно создавать или запоминать — ни логинов, ни паролей, ни профилей.

### 🛠️ Технологический стек

Приложение построено с упором на производительность, безопасность типов и современные практики фронтенд‑разработки.

- **Фронтенд**: [Svelte 4](https://svelte.dev/)
- **Язык**: [TypeScript](https://www.typescriptlang.org/)
- **Стили**: [Tailwind CSS](https://tailwindcss.com/)
- **Сборка**: [Vite](https://vitejs.dev/)

### 🚀 Быстрый старт

#### Требования

- Node.js (рекомендуется последняя LTS‑версия)
- npm или pnpm

#### Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/nicejade/life-spent.git
cd life-spent

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev

# Собрать production-версию
npm run build
```

### 🌐 Доступ

- **Локальная разработка**: выполните `npm run dev` и откройте `http://localhost:5173` в браузере.
- **Продакшн‑сборка**: выполните `npm run build`, затем задеплойте папку `dist/` на любой статический хостинг.
- **Хостинг**: подойдёт любой провайдер статического хостинга (Vercel, Netlify, GitHub Pages и т.п.).

### 📂 Структура проекта

```bash
src/
├── helper/            # Базовая логика, константы и утилиты
├── types/             # Определения типов TypeScript
├── components/        # UI‑компоненты Svelte
├── assets/icons/      # SVG‑иконки (check, github, moon, selector, sun)
├── styles/            # Минимальный глобальный стиль
├── App.svelte         # Корневой компонент приложения
└── main.ts            # Точка входа
```

### 🎨 Дизайн‑принципы

- **Сдержанность**: минимум текста, чёткая типографика, никаких лишних отвлекающих элементов.
- **Стеклянные карточки**: полупрозрачные панели и мягкие слои создают современное, немного церемониальное настроение.
- **Доступность**: семантический HTML и поддержка ARIA с самого начала разработки.
- **Адаптивность**: mobile‑first‑подход, чтобы можно было на минуту остановиться и взглянуть на время с любого устройства.

### 🤝 Участие в разработке

Мы открыты для аккуратных и продуманных изменений, которые сохраняют созерцательную атмосферу проекта.

- **Перед крупными изменениями**: откройте issue и обсудите идею и направление.
- **Рабочий процесс**: сделайте fork, создайте ветку от `main`, затем отправьте точечный, хорошо сфокусированный pull‑request.
- **Организация кода**: логика расчётов и бизнес‑правила — в `lib/`, пользовательский интерфейс — в компонентах `src/`.
- **При изменении логики**: по возможности добавляйте тесты или оставляйте структуру, к которой легко добавить тесты в будущем.
- **Стиль**: используйте utility‑классы Tailwind и избегайте избыточного кастомного CSS.
